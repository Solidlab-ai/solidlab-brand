# solidlab-brand

Living brand reference for **Solidlab** — deployed to **brand.solidlab.ai**.

## What this is

A publicly accessible, interactive brand system documenting how Solidlab looks,
sounds, and behaves. Complementary to (but separate from) the main marketing
site at solidlab.ai.

## Structure

- `index.html` — overview / landing page
- `preview/` — 20 interactive preview cards (colors, type, components, motion)
- `slides/` — 9-slide template deck
- `assets/` — brand guidelines PDF, logo SVGs
- `fonts/` — Geist + Geist Mono (variable)
- `colors_and_type.css` — design tokens

## Stack

Pure static HTML + CSS. No build step. No JavaScript framework.

## Deploy

Auto-deploys to Vercel on push to `main`.
