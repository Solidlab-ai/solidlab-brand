/ usage

The weight contrast between solid (Geist 700) and lab (Geist 400) IS the logo. No symbol beside the wordmark. All SVGs are outlined — no font dependency. Solid-background variants are for stamps and guaranteed legibility; transparent variants are for photos, overlays, and colored surfaces. For third-party use, contact hello@solidlab.ai. Do not modify weight ratio, tracking, or the .ai color.

/ installation

Open any *-signature.html file in Chrome. Select everything inside the signature box (Cmd+A), copy (Cmd+C), then paste into Gmail → Settings → Signature. Assign one signature per "Send mail as" address. For mobile Gmail, use the plain-text fallbacks — the mobile app doesn't support HTML signatures. The jens@ signature is transparently labeled as an AI assistant. The noreply@ signature redirects replies to hello@. The simba@ signature is a joke — and that's the point.

/ qr proxy system

All QR codes route through solidlab.ai/qr/* redirect endpoints. This means the QR destinations are editable anytime without reprinting — just update vercel.json in the solidlab-deploy repo. Routes: /qr/card-ulrik, /qr/card-solidlab, /qr/card-charlotte.

/ print specs

Format: 85×55mm (standard business card) with 3mm bleed. Material: Soft Touch + 400g premium matte. 8 SVG files — path-outlined vectors, font-independent. Vistaprint workflow: upload front + back as separate sides. Recommended first order: 100 Ulrik sort + 50 Ulrik antikkhvit + 50 Solidlab sort = 200 total.

Last opp 02-content-templates.md og 03-product-context.md som kunnskapsfiler. 4. Start ny chat og prøv: "LinkedIn-post om Bølgevarsel-lansering".

/ why claude project, not webapp

Fordi det virker umiddelbart uten kode, hosting eller vedlikehold. Når vi har 2 ukers bruksdata kan vi bygge studio.solidlab.ai som webapp med eksakte features du faktisk trenger. Start enkelt, iterer basert på bruk. Projects oppdateres automatisk — når vi forbedrer instruksjonene får du bedre output uten å bygge noe nytt.
