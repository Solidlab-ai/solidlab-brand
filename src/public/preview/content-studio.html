Last ned 01-project-instructions.md nedenfor og lim inn i prosjektets "Custom instructions"-felt.
/ step 03
Last opp kontekst
Last opp 02-content-templates.md og 03-product-context.md til prosjektet som knowledge files.

Test det
Start ny chat INNE I PROSJEKTET. Skriv "LinkedIn-post om Bølgevarsel" og se magien.
